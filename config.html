PROXMOX CONFIGURACION MAQUINA GITHUB WIKI



En este apartado explicamos como se instala y configura un contenedor una vez instalado el proxmox, con el objetivo de almacenar todos los archivos correspondientes a nuestra página en el mismo.

Tras haber instalado el proxmox lo primero que debemos hacer es acceder al mismo. Para ello primero hay que activar el servidor lampp a través de la terminal. Para ello debemos acceder a la terminal, y después introducir el comando "/opt/lampp/lampp start", haciendo que se inicialice el apache2, el mysql y el php.

Una vez activado todo debemos introducir lo siguiente en el apartado de las Url: https://(dirección ip):8006 (En nuestro caso la ip es 10.9.53.111), y accederemos a la página del proxmox.

Dentro de la página nuestra intención es la de crear un contenedor donde se almacenará todos los datos sobre el blog, y para ello necesitamos una plantilla, así que lo primero que debemos hacer es descargar una plantilla. Para ello debemos acceder al shell del proxmox situado donde mostramos en la imagen, y una vez dentro tecleamos en la terminal "pveam update" para actualizar las plantillas, y seguido "pveam available" para ver las plantillas disponibles para descarga. Escogemos la plantilla a descargar e introducimos el comando "pveam download local (nombre de la plantilla a descargar)"

Ahora toca instalar el contenedor. Hay un botón arriba a mano derecha en el que pone "Crear CT". Lo pulsamos. Aparecerá una ventana llamada general con información a rellenar para la instalación. Nosotros rellenaremos los apartados de nodo, CT y contraseña, el resto no lo necesitamos. En la ventana de plantilla omitimos el almacenamiento y como plantilla seleccionamos la que acabamos de descargar en el apartado anterior. En la siguiente parte como tamaño de disco ponemos 16 y seguimos para adelante. En la CPU pondremos dos nucleos, aunque no es determinante para lo que vamos a hacer una vez instalada la maquina. En la memoria ponemos 8000 en ambos apartados y continuamos. Y aquí aparece un apartado de suma importancia para llevar a cabo el reto del blog, el de redes. Debemos introducir la ip a utilizar en el contenedor y la puerta de enlaze. Nosotros utilizaremos la ip 10.9.52.106/23 y la puerta de enlace 10.9.52.100. El DNS lo omitimos, no lo necesitaremos. Y por último aparece una ventana de confirmación con todos los datos que hemos introducido, donde decidiremos si queremos en efecto realizar la instalación o no, donde seleccionaremos "Finalizado".

El contenedor ya está instalado, ahora toca configurar las herramientas que necesitaremos utilizar. Primero instalaremos las net-tools, para poder hacer uso del comando "ifconfig", por ejemplo. Acto seguido confirmaremos que la red que estamos usando es la correcta.
